# 📚 Padhlo – Collaborative Learning Platform

Padhlo is a full-stack MERN application designed to help students collaborate, manage tasks, track progress, and communicate effectively within study groups and classrooms.

---

## 🌐 Live Demo

🔗 https://padhlo-six.vercel.app/login

---

## 🚀 Features

### 👤 Authentication

* Secure JWT Authentication
* User Registration & Login
* Protected Routes

### 📊 Dashboard

* Personalized Dashboard
* Task Statistics
* Study Streak Tracking
* Progress Overview

### ✅ Task Management

* Create Tasks
* Complete Tasks
* Delete Tasks
* Deadline Tracking
* Streak Updates on Completion

### 🏫 Classroom Management

* Create Classroom
* Join Classroom via Room Code
* Leave Classroom
* Delete Classroom (Creator Only)

### 📝 Classroom Tasks

* Assign Tasks to Classroom Members
* Track Task Completion
* View Completed Members

### 📈 Progress Tracking

* Monitor Member Progress
* Completion Analytics
* Individual Contribution Tracking

### 🏆 Leaderboard

* Global Streak Leaderboard
* Classroom Performance Leaderboard

### 📂 Notes & Resources

* Upload Notes
* Share Learning Resources
* Access Classroom Materials

### 📅 Calendar

* Personal Events
* Study Schedule Tracking
* Task Deadlines Visualization

### 💬 Real-Time Classroom Chat

* Instant Messaging with Socket.IO
* Edit Own Messages
* Delete Own Messages
* Real-Time Updates

### 👤 Profile Management

* Upload Profile Picture
* Update Personal Information
* Skills & Goals Tracking

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router
* Tailwind CSS
* Zustand
* Axios
* React Hot Toast
* Socket.IO Client

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer
* Cloudinary
* Socket.IO

### Deployment

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas
